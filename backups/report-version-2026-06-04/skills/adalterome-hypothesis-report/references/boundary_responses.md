# Boundary Responses

## Hypothesis Not Found

Say that no records were returned for the hypothesis. Suggest listing hypotheses with `adalterome-api hypotheses`.

## Ambiguous Hypothesis

If the query matches multiple combined labels, report this explicitly and describe the result as a broad support map.

## Sparse Evidence

If few sentence-level rows are returned, provide a concise source-traceable note rather than a deep synthesis.

## Weak Support

If support comes mostly from generic sentences or curated explanations, avoid strong biological claims.
