# Deep Hypothesis Report Contract

Use this structure:

1. Query scope and hypothesis definition
2. Executive support claim
3. Top supporting genes and terms
4. Evidence and source map
5. Original evidence traces
6. Support pattern synthesis
7. Evidence strength and limitations
8. Follow-up analysis priorities

## Required Standards

- Include the exact hypothesis query and request URLs.
- Include top genes and terms from `/hypothesis/overview`.
- Include original evidence sentences and PubMed links from `/hypothesis/support`.
- Include evidence and quality scores.
- Separate primary sentence evidence from `HypothesisReason` and `ExtendedExplanation`.

## Interpretation Rules

- Do not say the hypothesis is proven.
- Use repeated genes, terms, and PMIDs to describe support patterns.
- Mention combined hypothesis labels when a record links multiple hypotheses.
